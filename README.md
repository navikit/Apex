# Apex

This repository contains a collection of small, useful programs and experiments.

## What this repo is for
- A place to store **random useful utilities** and **demo projects**.
- A showcase of code snippets, small tools, and quick hacks that can be reused or extended.

## How to use
- Browse the folders to find tools (e.g., `pi_digits/` for the π digit generator).
- Feel free to fork, extend, and build on top of anything in this repo.

## Why follow
If you like the types of utilities here, follow the repo for ongoing updates, new mini-projects, and quick code experiments.
